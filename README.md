## Usage
Seed database:
```sh
npm run seed
```
Start server:
```sh
npm run start:dev
```
Watch/Build with Webpack:
```sh
npm run build:dev
```
### Installing Dependencies

From within the root directory:

```sh
npm install -g webpack
npm install
```
